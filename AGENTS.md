# AGENTS Rules For This Repository

## Project Constraints

- Проект строго статический: не добавлять сборку, `npm`, bundler и внешние зависимости без явной просьбы пользователя.
- Перед коммитом обязательно выполнять локальную проверку через HTTP-сервер в браузерном режиме.
- При правках UI не ломать RU/EN локализацию, фильтры классов и поведение карты.
- Сохранять и поддерживать маркеры `START_BLOCK_*` / `END_BLOCK_*` в коде.

## Skill Assignment Policy

- Назначать скиллы автоматически под этап задачи.
- Для итеративной проверки и улучшения применять `iterative-retrieval` как обязательный протокол:
  1. Проверка структуры и целостности путей.
  2. Внесение правок.
  3. Регрессионная проверка UI/локализации/фильтров.
